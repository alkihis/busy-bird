Class _PageManager

Gère les pages de l'application. Utilisée pour gérer le système de pile de pages. Pour pousser une nouvelle page sur les autres, utilisez push(). Pour revenir à la page précédente, utilisez back(). Vous pouvez aussi utiliser pop() qui ne fait pas de vérification d'usage lors du dépop. Cette classe ne doit être instanciée qu'une seule fois !
