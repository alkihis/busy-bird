Class FormSchemas

Contient les différents schémas de formulaire, le schéma actuellement chargé, et charge automatiquement au démarrage, depuis le serveur ou depuis le stockage interne, les schémas disponibles.

Après création de l'objet, lancez l'initialisation avec init() puis attendez la fin de l'initialisation avec onReady().
